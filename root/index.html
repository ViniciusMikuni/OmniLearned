OmniLearned

A Foundation Model Framework for All Tasks Involving Jet Physics

GitHub Repo | arXiv Paper

OmniLearned Model

OmniLearned is a Point Cloud Foundation Model for Jet Physics. It builds upon OmniLearn with improved architecture and pre-trained on 1 Billion jets with all datasets acessible from the software package!

Results Showcase

Jet Tagging Performance

We compare OmniLearned results for the Community Top Tagging Dataset and that for a fixed signal efficiency, OmniLearned achieved the highest background rejection of all benchmarks!

ATLAS Flavour Tagging

We also see improvements in Flavour Tagging using the ATLAS open dataset consisting of realistic simulations, sometimes improving the background rejection by a factor 2 for some signal efficiencies!

Anomaly Detection

Even though top quarks are 0.1% of the Aspen Open Jets dataset, we are able to find top quarks with OmniLearned even if the model trained from scratch cannot!.
