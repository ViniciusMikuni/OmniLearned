OmniLearned

A Foundation Model Framework for All Tasks Involving Jet Physics

GitHub Repo | arXiv Paper

OmniLearned Model

OmniLearned is a Point Cloud Foundation Model for Jet Physics. It builds upon OmniLearn with improved architecture and is pre-trained on 1 billion jets, with all datasets accessible from the software package!

Results Showcase

Jet Tagging Performance

We compare OmniLearned results for the Community Top Tagging Dataset and find that for a fixed signal efficiency, OmniLearned achieves the highest background rejection of all benchmarks!

ATLAS Flavour Tagging

We also see improvements in flavour tagging using the ATLAS open dataset, consisting of realistic simulations — sometimes improving the background rejection by a factor of two for certain signal efficiencies!

Anomaly Detection

Even though top quarks are only 0.1% of the Aspen Open Jets dataset, OmniLearned can still find them — while models trained from scratch cannot!
